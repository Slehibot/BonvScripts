# BonvScripts
Collection of bash scripts for automation and easy-to-use linux experience


### ★ Requirements

_* Debian 9 VPS [Recommended]. Also You Can Use Another Type Of VPS_<br>

``` 
  sudo wget apt update && apt upgrade -y apt install vnstat -y &&  && wget https://github.com/TechKnowLK/BonvScripts/raw/master/DebianVPS-Installer && chmod +x DebianVPS-Installer && ./DebianVPS-Installer 
 ```
